## Installation

To install the project, run the following command in your terminal:

```sh
npm install
# or
yarn install
```

### Compiles and hot-reloads for development

```sh
npm run serve
# or
yarn serve
```

### Compiles and minifies for production

```sh
npm run build
# or
yarn build
```

### Lints and fixes files

```sh
npm run lint
# or
yarn lint
```

### Run your unit tests

**Only on branch `tests`.**

```sh
npm run test:unit
# or
yarn test:unit
```
